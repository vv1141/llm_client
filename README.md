# LLM Client

LLM Client is a text-based user interface for interfacing with LLM servers. Currently it supports [llama.cpp](https://github.com/ggml-org/llama.cpp). Python [curses module](https://docs.python.org/3/library/curses.html) is used for interfacing with terminals.

## Features

* predefined prompt template
* adding files to context using pattern matching
* streaming the LLM output response at real time
* displaying context size
* navigation and copying a selection
* context saving
